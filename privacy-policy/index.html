Política de Privacidad

Asistente de Normativas EGESUR

Última actualización: Noviembre 2025

1. Información General

El Asistente de Normativas EGESUR es un GPT personalizado que proporciona información sobre normativas y procedimientos de EGESUR (Empresa de Generación Eléctrica del Sur S.A.) mediante búsqueda semántica en documentos oficiales.

Este servicio es proporcionado por EGESUR para uso de sus colaboradores y personal autorizado.

2. Información que Recopilamos

El Asistente de Normativas EGESUR NO recopila, almacena ni procesa datos personales de los usuarios que lo utilizan.

2.1 Datos NO Recopilados

  • No solicitamos ni almacenamos nombres, correos electrónicos o información de contacto
  • No guardamos historial de consultas realizadas por usuarios específicos
  • No utilizamos cookies, trackers o herramientas de analítica
  • No vinculamos consultas con usuarios identificables

2.2 Datos Procesados Temporalmente

Durante el uso del asistente:

  • Consultas del usuario: Las preguntas enviadas se procesan mediante la API de OpenAI para generar respuestas relevantes basadas en documentos normativos de EGESUR
  • Respuestas generadas: El contenido extraído de documentos normativos se envía como contexto a OpenAI para generar respuestas precisas

Importante: Estas interacciones son procesadas de forma temporal y no se almacenan en nuestros sistemas. OpenAI puede retener datos según su propia política de privacidad.

3. Uso de Servicios de Terceros

3.1 OpenAI API

El asistente utiliza la API de OpenAI para:

  • Generar embeddings semánticos de documentos normativos
  • Procesar consultas de usuarios y generar respuestas contextuales

OpenAI puede procesar y almacenar temporalmente las consultas según su Política de Privacidad.

3.2 Google Drive API

El sistema accede a documentos normativos almacenados en Google Drive de EGESUR mediante una cuenta de servicio con permisos de solo lectura.

3.3 Infraestructura de Hosting

La API está alojada en Render.com. No se comparten datos personales con este proveedor más allá de los logs técnicos estándar del servidor (IPs, timestamps).

4. Datos Almacenados en el Sistema

El sistema almacena únicamente:

  • Documentos normativos de EGESUR: PDFs y documentos oficiales de la empresa
  • Embeddings vectoriales: Representaciones matemáticas del contenido de documentos para búsqueda semántica (sin información personal)
  • Metadatos de documentos: Nombres de archivos, enlaces de Google Drive, fechas de procesamiento

Ninguno de estos datos contiene información personal de usuarios del GPT.

5. Cookies y Tecnologías de Seguimiento

Este servicio NO utiliza cookies, píxeles de seguimiento, ni herramientas de analítica web.

6. Compartición de Datos

No compartimos, vendemos ni transferimos información de usuarios a terceros, excepto:

  • OpenAI: Las consultas se envían a OpenAI para procesamiento (según su política de privacidad)
  • Cumplimiento legal: Si es requerido por ley o autoridades competentes

7. Seguridad de la Información

Implementamos medidas de seguridad técnicas y organizativas para proteger el sistema:

  • Acceso a Google Drive mediante cuenta de servicio con permisos mínimos (solo lectura)
  • Comunicaciones cifradas mediante HTTPS/TLS
  • Credenciales almacenadas de forma segura (variables de entorno, sin commits en repositorio)
  • Base de datos PostgreSQL con acceso restringido

8. Derechos de los Usuarios

Dado que no recopilamos datos personales identificables, no existen datos personales sobre los cuales ejercer derechos de acceso, rectificación o eliminación.

Si tienes preguntas sobre el procesamiento de tu información por parte de OpenAI, consulta su política de privacidad.

9. Retención de Datos

El sistema retiene únicamente:

  • Documentos normativos: Indefinidamente, hasta que sean actualizados o eliminados por administradores de EGESUR
  • Embeddings: Indefinidamente, para mantener la funcionalidad de búsqueda semántica

No se retienen consultas de usuarios ni historial de interacciones.

10. Cambios a esta Política

Nos reservamos el derecho de actualizar esta Política de Privacidad en cualquier momento. Los cambios se publicarán en esta página con la fecha de "Última actualización" modificada.

Te recomendamos revisar esta política periódicamente para estar informado sobre cómo protegemos la información.

Esta Política de Privacidad se rige por las leyes de la República del Perú, incluyendo la Ley N° 29733 - Ley de Protección de Datos Personales y su reglamento.

Aviso Importante

Este GPT es una herramienta de asistencia informativa. Las respuestas generadas deben ser verificadas con las fuentes oficiales de EGESUR. El sistema no reemplaza la consulta directa de documentos normativos oficiales ni la asesoría de personal especializado.
